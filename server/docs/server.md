# Документация к API-серверу

## Быстрый старт

```bash
npm i
npm run dev
```

## Используемый пакет модулей в сервере

```json
...,
"dependencies": {
    "cors": "^2.8.5",
    "dotenv": "^17.2.3",
    "express": "^5.1.0"
  },
...,
```

### Краткое описание используемых модулей

- **express** — основной фреймворк для создания API и работы с роутами.
- **cors** — позволяет разрешить запросы с других доменов (Cross-Origin Resource Sharing).
- **dotenv** — загружает переменные окружения из файла `.env` в `process.env` для удобной конфигурации.
